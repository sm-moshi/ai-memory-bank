# Progress Index

This file provides a summary and entry point for all project progress tracking. Use the links below to navigate to detailed sections:

- [Current Progress](./current.md):
  - What is actively being worked on, current status, known issues, and next steps.
- [Progress History](./history.md):
  - Log of completed work, milestones, and major achievements.

## How to Use

- Start here to get an overview of the project's progress system.
- Check `current.md` for up-to-date work and actionable items.
- Review `history.md` for a chronological record of what has been accomplished.

*This modular structure helps keep progress tracking clear, maintainable, and easy to query for both humans and AI agents.*
