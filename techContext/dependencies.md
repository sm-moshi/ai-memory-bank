# Dependencies

This project relies on the following key dependencies for its backend (Node.js/extension) and frontend (webview):

## Backend (Extension)
- **@modelcontextprotocol/sdk** — MCP protocol and memory bank tools
- **express** — Lightweight HTTP server for MCP endpoints
- **cors** — Cross-origin resource sharing middleware
- **zod** — Schema validation and parsing

## Frontend (Webview)
- **react** — UI components
- **react-dom** — React DOM rendering
- **react-icons** — Icon library for consistent UI
- **@tailwindcss/vite** — Tailwind CSS integration for Vite
- **tailwindcss** — Utility-first CSS framework
- **@vscode-elements/elements** — VS Code UI elements
- **@vscode-elements/react-elements** — React bindings for VS Code UI elements

## Dev & Tooling
- **esbuild** — Fast bundler for extension backend
- **vite** — Bundler/dev server for webview
- **typescript** — Type safety and maintainability
- **eslint** — Linting
